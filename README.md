# Paradise Nursery Shopping Cart Application 🌿

A full-featured React + Redux e-commerce app for an online plant shop.

## Features
- 3+ categories (Indoor, Succulents, Flowering)
- 6+ plants per category (18 total)
- Thumbnails, names, prices
- Add to cart with disabled button
- Cart count in navbar (dynamic)
- Increase/decrease quantity
- Remove items
- Total cost calculation
- Routing (Home, Plants, Cart, About)
